Calculate squared error regression loss.

Usage

squared_error(y_true, y_pred)

Arguments

y_true
Ground truth (correct) target values.
y_pred
Estimated target values.

Value

A numeric vector of length equal to max(length(y_true), length(y_pred)).

Description

Calculate squared error regression loss.

Examples

squared_error(1:10, 10:1)


 [1] 81 49 25  9  1  1  9 25 49 81
